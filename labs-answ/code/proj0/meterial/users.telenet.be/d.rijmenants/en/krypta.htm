KRYPTA is a basic tool to encrypt text from KRYPTA to the clipboard or to decrypt the clipboard content to the KRYPTA text window. It is mainly designed to offer easy-to-use encryption to protect your e-mail communication. This small program provides a basic but strong encryption in a practical user-interface. KRYPTA is therefore a good alternative for people that are not familiar with cryptography or don't want to use complex encryption software. A complete description of the symmetric encryption algorithm is provided in the Readme and is easy to implement, even by novice programmers, thus making it easy to apply in various compatible software applications in different programming languages and on different platforms.

Working with KRYPTA could not be easier. If you want to send an encrypted message by e-mail, just start KRYPTA and enter the text that you want to encrypt. Click the 'Encrypt' menu and you will be prompted to enter your secret key. The message is automatically encrypted and sent to the clipboard. Just paste (CTRL+V) the clipboard to your e-mail program and your mail is ready to go. If you received an encrypted message, just select the ciphertext and copy it (CTRL+C) to the clipboard. Start KRYPTA and hit the 'Decode' menu. the decrypted text will appear in the KRYPTA text box after entering the correct key. The original message is never stored on your computer and the clipboard will only hold encrypted data. All message copies, stored in e-mail folders or other locations, will be encrypted versions and required the correct password to read them. Of course, thanks to the clipboard interface, KRYPTA can be used with any other text based program.

Version 2.1.0 incorporates several new options. You can choose between hexadecimal, base64 and letters-only encryption output. Customization of the crypto algorithm enables the creation of an exclusive user group.

Runs on Windows™ 98/ME/2000/XP/Vista/Win7/Win 8 and on MAC with CrossOver, Parallels Desktop or WINE (open source).
